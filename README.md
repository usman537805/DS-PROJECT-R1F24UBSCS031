Project Description

This Restaurant Management System is a Data Structures project developed in C++. It uses a Queue to manage pending customer orders using the First-In-First-Out (FIFO) principle and a Stack to maintain the history of completed orders using the Last-In-First-Out (LIFO) principle. The system allows users to add, search, display, serve, and count orders efficiently.
